# Техническое задание (ТЗ)

## 1. Название проекта
Team LMS — платформа для загрузки и проверки учебных работ.

## 2. Цель проекта
Создать веб-приложение, позволяющее студентам регистрироваться, загружать файлы заданий и получать оценку.

## 3. Функциональные требования
1. Регистрация и авторизация пользователей.
2. Личный кабинет с формой загрузки файлов.
3. Сохранение информации о работах в БД.
4. Просмотр последних работ на главной странице.
5. REST API для интеграций и проверки.

## 4. Нефункциональные требования
- Код оформлен по PEP 8.
- Структура разделена на модули.
- Используется `requirements.txt`.
- Ограничение размера файла: до 5 МБ.

## 5. Используемые технологии
- Python, Flask
- Flask-SQLAlchemy (ORM)
- Flask-Login
- Flask-WTF
- HTML, CSS, Bootstrap
- SQLite

## 6. Этапы реализации
1. Подготовка архитектуры и БД.
2. Реализация авторизации.
3. Реализация загрузки файлов.
4. Реализация API.
5. Тестирование и подготовка защиты.
