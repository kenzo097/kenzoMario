# Team LMS (Flask Project)

Учебный веб-проект для групповой работы (2-3 участника).

## Возможности

- Регистрация и авторизация пользователей
- Загрузка файлов работ
- Отдельный кабинет преподавателя для проверки и выставления баллов
- Чат обратной связи между учеником и учителем по каждой работе
- Хранение данных в SQLite через ORM (SQLAlchemy)
- REST API для просмотра и оценки работ
- Bootstrap-интерфейс

## Быстрый старт

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python run.py
```

Откройте `http://127.0.0.1:5000`.

## Доступ преподавателя

По умолчанию преподавателем считается пользователь с email `teacher@example.com`.
Список email преподавателей задаётся в `app/__init__.py` в параметре `TEACHER_EMAILS`.

## Чат обратной связи

- Ученик открывает чат из личного кабинета кнопкой `Чат с преподавателем`
- Учитель открывает чат из раздела `Проверка работ` кнопкой `Открыть чат`
- Чат привязан к конкретной загруженной работе

## API

- `GET /api/submissions` — список работ
- `GET /api/submissions/<id>` — одна работа
- `POST /api/submissions/<id>/score` — выставление балла

Пример тела запроса:

```json
{
  "score": 85
}
```

## Структура

- `app/` — исходный код Flask-приложения
- `docs/` — материалы для сдачи проекта
- `requirements.txt` — зависимости
