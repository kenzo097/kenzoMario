# Super Mario Game (Arcade + PyQt6)

Игра Марио, написанная на Python с использованием библиотеки Arcade для игрового движка и PyQt6 для дизайна интерфейса.

## Установка

1. Установите Python 3.7 или выше
2. Установите зависимости:
```
pip install -r requirements.txt
```

## Запуск

```
python main.py
```

## Управление

- **Стрелки влево/вправо** или **A/D** - движение
- **Пробел** или **Стрелка вверх** или **W** - прыжок
- **ESC** - выход из игры

## Особенности

- ✅ Стартовое окно с красивым дизайном на PyQt6
- ✅ Финальное окно (победа/проигрыш) на PyQt6
- ✅ Подсчет и вывод результатов
- ✅ Спрайты через Arcade
- ✅ Система коллизий (Arcade PhysicsEngine)
- ✅ Анимация персонажей
- ✅ 3 уровня сложности
- ✅ Камера, следующая за игроком
- ✅ Система частиц для эффектов
- ✅ Звуковая система (готова для добавления звуков)
- ✅ Физический движок (Arcade PhysicsEngine)
- ✅ Сохранение рекордов в data.txt

## Структура проекта

- `main.py` - главное меню на PyQt6
- `game_window.py` - окно игры
- `arcade_game.py` - игровой движок на Arcade
- `player_arcade.py` - класс игрока
- `level_arcade.py` - система уровней
- `requirements.txt` - зависимости

## Технологии

- **Arcade** - игровой движок для 2D игр
- **PyQt6** - современный GUI фреймворк для дизайна интерфейса

## Цель игры

Соберите монеты, избегайте врагов (или прыгайте на них), дойдите до конца уровня!
